## Overview

This project was created to demonstrate my ability to design and develop a modern restaurant landing page using WordPress.

The website emphasizes visual impact, storytelling, responsive design, and an intuitive user experience that encourages visitors to explore the menu and visit the restaurant.

---

## Design Concept

The design aims to create a premium Japanese ramen brand image.

Special attention was paid to:

- Strong first impression
- Appetite appeal
- Warm atmosphere
- Easy navigation
- Mobile-first experience
